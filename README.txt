\\ToScrap
This project scraps all the citation and author on the website "https://quotes.toscrape.com/" 
Then it creates the citation.txt file with the collection of citations and authors found.


\\IMDB
Lists the 250 top movies on the IMDB and create a CSV file with name, classification, releasing date of the movie.
This project gets all the links on the first page and open them to get the classification
